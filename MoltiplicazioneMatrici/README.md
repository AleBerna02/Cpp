# Moltiplicazione di due matrici

Scrivere un programma che chiede e legge le dimensioni di due matrici compatibili per la moltiplicazione (le colonne della prima uguali alle righe della seconda), chiede e legge i loro elementi, ne calcola il prodotto e stampa la matrice risultante.

Il programma deve essere scritto in un file chiamato `matrixmult.cpp`.
