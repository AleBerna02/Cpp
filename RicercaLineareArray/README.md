# Ricerca lineare in un array

Scrivere un programma che definisce una funzione per cercare un numero intero all'interno di un array. La funzione deve restituire l'indice della prima occorrenza del numero, oppure -1 se il numero non è presente. Il main legge la dimensione, riempie l'array e chiede il numero da cercare.

Il programma deve essere scritto in un file chiamato `linearsearch.cpp`.
