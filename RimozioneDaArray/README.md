# Rimozione di un elemento da un array

Scrivere un programma che definisce una funzione per rimuovere un elemento da un array in una posizione specifica, spostando all'indietro gli elementi successivi per coprire il buco. Il main legge l'array e l'indice da eliminare, stampando poi l'array aggiornato.

Il programma deve essere scritto in un file chiamato `deletearray.cpp`.
