# Esercizio 2: Somma con messaggi di richiesta

Scrivere un programma che esegue la somma di due interi, aggiungendo messaggi di richiesta per l'utente prima di ogni input.

Il programma deve stampare un messaggio del tipo *"Inserisci il valore di..."* prima di leggere ogni variabile.

**File richiesto:** `asksum.cpp`
