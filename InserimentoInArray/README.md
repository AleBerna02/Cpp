# Inserimento di un elemento in un array

Scrivere un programma che definisce una funzione per inserire un nuovo numero intero in un array in una posizione specifica, spostando in avanti gli elementi successivi. Il main legge i dati, l'indice di inserimento e il nuovo valore, per poi stampare l'array aggiornato.

Il programma deve essere scritto in un file chiamato `insertarray.cpp`.
