# Ricerca in una matrice

Scrivere un programma che legge una matrice di interi (righe, colonne e valori) e un numero da cercare. Il programma deve stampare le coordinate (riga e colonna) della prima occorrenza del numero. Se il numero non è presente, deve stampare "-1 -1".

Il programma deve essere scritto in un file chiamato `searchmatrix.cpp`.
