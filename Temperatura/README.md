# Classificazione temperatura

Scrivere un programma che legge da input un numero intero `Temp` e stampa un messaggio basato sull'intervallo termico in cui ricade (da "Freddo dannato" a "Caldo da morire"). Gestisce anche i casi estremi segnalando la possibile rottura del termometro.

Il programma deve essere scritto in un file chiamato `temperature.cpp`.
