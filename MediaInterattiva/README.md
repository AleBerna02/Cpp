# Media interattiva

Scrivere un programma che chiede all'utente di inserire e leggere numeri interi e poi chiede all'utente se vuole continuare e legge la risposta finché l'utente non risponde di no. Finito il ciclo di lettura stampa la media dei numeri letti.

Il programma deve essere scritto in un file chiamato `mediainterattiva.cpp`.
