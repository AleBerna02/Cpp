# Media di valori reali

Scrivere un programma che legge un certo numero di valori reali e ne stampa la media. Se il numero di valori richiesto non è strettamente positivo, il programma deve terminare con un codice di errore. 

L'esercizio richiede tre implementazioni differenti:
1. `mediafor.cpp` (utilizzando un ciclo for)
2. `mediawhile.cpp` (utilizzando un ciclo while)
3. `mediadowhile.cpp` (utilizzando un ciclo do-while)
