# Grafico a barre con asterischi

Scrivere un programma che ripete le seguenti operazioni finché l'utente non decide di terminare: chiede all'utente un numero intero positivo, stampa su una nuova riga tante '*' quanto è grande il numero, chiede all'utente se vuole terminare (S o s).

Il programma deve essere scritto in un file chiamato `barplot.cpp`.
