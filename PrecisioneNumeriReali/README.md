# Precisione dei numeri reali

Scrivere un programma che chiede all'utente un numero reale e lo legge. Quindi, in cascata, lo divide per 4.9, per 3.53 e per 6.9998. Poi, sempre in cascata, moltiplica per 4.9, per 3.53 e per 6.9998. Infine confronta il risultato ottenuto con il numero iniziale e se rappresentano due numeri reali diversi stampa "moltiplicare NON è l'inverso di dividere".

Il programma deve essere scritto in un file chiamato `checkprecision.cpp`.
