# Lettura dell'ora in italiano

Scrivere un programma che scrive in lettere i nomi italiani delle ore, approssimati per difetto a 15 minuti. Il programma deve prendere in input due valori interi (ore e minuti) e validare l'input. Se corretto, scrive "Sono le ore" seguito dal valore delle ore in lettere e dal valore dei minuti approssimato al quarto d'ora. Se i minuti non sono divisibili esattamente per 15, scrive " circa".

Il programma deve essere scritto in un file chiamato `saytime.cpp`.
