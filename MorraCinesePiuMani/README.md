# Morra cinese a più mani

Scrivere un programma che gioca più mani di morra cinese. Alla fine di ogni mano deve chiedere all'utente se vuole continuare; se l'utente risponde 'S' o 's' prosegue con un'altra mano, altrimenti termina.

Il programma deve essere scritto in un file chiamato `partitamorra.cpp`.
