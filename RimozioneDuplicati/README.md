# Rimozione dei duplicati

Scrivere un programma che definisce una funzione per rimuovere i valori duplicati da un array di interi, mantenendo solo la prima occorrenza di ciascun elemento e riducendo la dimensione dell'array. Il main legge i dati, richiama la funzione e stampa l'array pulito.

Il programma deve essere scritto in un file chiamato `removedup.cpp`.
